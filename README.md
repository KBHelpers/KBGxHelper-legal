## Privacy Policy

By using KBGxHelper, you agree to this Privacy Policy.

### Data the bot stores

KBGxHelper stores:

* Discord user ID.
* Current Discord username or display name.
* Galaxy/GX module data.
* Gatherinator stats.
* Helper settings, presets, filters, cleanup rules, and protected module flags.

### Message content access

KBGxHelper can read messages it can see. This is necessary to read EPIC RPG Galaxy/GX module messages, such as module inventory, produce results, and Gatherinator information.

Message content is used only to provide helper features, calculations, scan summaries, and cleanup suggestions.

### How data is used

Stored data is used only for KBGxHelper features, including module tracking, build calculations, cleanup suggestions, settings, and scan results.

KBGxHelper does not sell user data.

### Data deletion

You can delete your stored KBGxHelper data by using:

`/gunregister`

This deletes your profile, modules, Gatherinator data, settings, filters, cleanup rules, and protected flags.
